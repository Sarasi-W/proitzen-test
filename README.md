## Proitzen Test

### Instructions

```
git clone git@github.com:Sarasi-W/proitzen-test.git
cd proitzen-test
cp .env.example .env
php artisan key:generate
```

Update .env file with database configuration


```
composer update
npm install
npm run dev
```

checkout to dev branch and get a pull

```
php artisan migrate
php artisan db:seed
php artisan serve
```

```
username: admin@admin.com
password: password
```
